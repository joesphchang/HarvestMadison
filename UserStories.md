# User Stories for Harvest Madison
---
### User Authentication & Account Management
    - Sign up *
    As a new user, I want to create an account so that I can build a personal digital cookbook.

    - Sign In *
    As a registered user, I want to log in so that I can access my private recipes from any device.

    - Password Reset *
    As a registered user, I want to be able to reset my password.

### Recipe Discovery & Filtering
    - Find Recipes by Name *
    As a visitor, I want to search for recipes by title (e.g., "Spicy Basil Chicken") so I can quickly find what I want to cook.
    
    - Filter Recipes by Dietary Attributes *
    As a visitor, I want to filter recipes by attributes (e.g., Vegan, Gluten-Free, Under 30 mins) so I can find meals that fit my needs.

### Recipe Information & Viewing
    - View Recipe Details *
    As a visitor, I want to view ingredients, step-by-step instructions, and prep time so I can decide if I have the resources to cook it.

    - View Nutrition Information *
    As a visitor, I want to see the caloric and macro-nutrient breakdown for a recipe.

### Recipe Contributions (CRUD)

    - Add Personal Recipe *
    As a registered user, I want to manually input my own recipes (title, ingredients, steps) to keep them organized in one place.

### Admin & Data Management
    - Admin Manage Global Recipes *

## Non-MVP User Stories

### Personalization & Growth
    - Favorite/Save Recipes
    As a registered user, I want to "save" recipes from the global database to my profile for quick access.
    
    - Scale Servings
    As a user, I want to toggle the serving size (e.g., 2x or 0.5x) and have the ingredient quantities update automatically.

### Mobile & Kitchen Utility
    - "Cooking Mode"
    As a user, I want a mobile-optimized view that keeps the screen awake and shows one step at a time so I don't get flour on my phone.

    - Ingredient Checklist
    As a user, I want to cross off ingredients as I shop or prep so I don't miss a crucial component.

### Social & Engagement
    - Rate and Review
    As a user, I want to leave a star rating and a comment (e.g., "Added more garlic!") on public recipes to help other cooks.

    - Photo Uploads
    As a user, I want to upload a photo of my finished dish to the recipe page.
