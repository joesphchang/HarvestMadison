User Stories for Harvest Madison
--- 

### User Authentication & Account Management

    - Sign up *
    As a new user, I want to create an account so that I can build a personal digital cookbook.

    - Sign In *
    As a registered user, I want to log in so that I can access my private recipes from any device.

    - Sign Out *
    As a user, I want to securely log out of my account to ensure my session is terminated on shared devices.

    - Password Reset *
    As a registered user, I want to be able to reset my password.

### Recipe Discovery & Seasonality (The "Harvest" Core)

    - View Monthly Hero Ingredient *
    As a visitor, I want to see the "Ingredient of the Month" (e.g., Asparagus in May) on the homepage 
    so I can cook with what is currently in harvest in Madison.

    - View Seasonal Recipe Collection *
    As a visitor, I want to see a randomized list of recipes featuring the current month's hero 
    ingredient so I have variety in my seasonal cooking.

    Find Recipes by Name *
    As a visitor, I want to search for recipes by title so I can quickly find what I want to cook.

    Filter Recipes by Dietary Attributes *
    As a visitor, I want to filter recipes by attributes to find meals that fit my needs.

### Recipe Information & Viewing

    - View Recipe Details *
    As a visitor, I want to view ingredients, step-by-step instructions, and prep time.

    - View About Page *
    As a visitor, I want to view an About page to understand the project mission and 
    how seasonal ingredients are selected for the Madison area.

    - View Nutrition Information *
    As a visitor, I want to see the caloric and macro-nutrient breakdown for a recipe.
